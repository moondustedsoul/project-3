# Reflection

I found TDD to be an interesting way to approach completing a coding project. Having the exact guidelines for each method in the tests  and working from there was a lot more effective than starting code without any sort of guidelines to follow. I also recognized the pattern in the code for each method fairly quickly, so the implementation of most of the methods felt much simpler than expected, especially with the recommendations from ChatGPT to follow.

I think that this assignment had a satisfying blend of independent coding and AI support. With ChatGPT providing an outline to follow during implementation, I could solve each problem presented by the tests myself, while still having the outline to guide me to the next steps. Additionally, I found ChatGPT's code review to be effective while avoiding simply giving me all the answers.

The workflow introduced in this project, and more specifically the AI aspects of it, proved favorable for me over our other projects thus far. I much preferred being able to create the code myself while simply receiving support from ChatGPT, and it seems to have created the highest-quality from any of the projects thus far.
